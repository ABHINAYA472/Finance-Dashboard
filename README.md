# Finance Dashboard

## Overview
A React-based Finance Dashboard to track income and expenses with charts and insights.

## Features
- Dashboard summary (Balance, Income, Expense)
- Charts (Line + Pie)
- Transactions table
- Add & delete transactions
- Role-based UI (Admin / Viewer)

## Tech Stack
- React (Vite)
- CSS
- Chart.js

## Run Locally
npm install
npm run dev
